
This application is developed using Visual Studio .NET Framework (C# Language) with Standard Chart-Component 

You can edit it on your system by using any version of Visual Studio .Net from 2010 and above ...

To open the application project:
 
 - Extract the MCLSim.zip file
 - Open the extracted folder "MySim3" then Run The file MySim1.sln
 - Add Reference for "Microsoft.Office.Interop.Excel" from .Net Tab
 - Rebuild and Run Application 

--No need for any specific configuration .. 

If just you need to run the application then, 
-- Run the exe file direclty in this path MCLSim\MySim\bin\Debug\MySim.exe
